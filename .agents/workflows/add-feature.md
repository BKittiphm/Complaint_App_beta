---
description: ขั้นตอนการเพิ่มฟีเจอร์ใหม่และอัปเดตไปยัง Mobile
---

1. สรุปความต้องการของฟีเจอร์ใหม่ให้ USER ยืนยันก่อนเริ่ม
2. แก้ไขโค้ดใน `src/` เพื่อเพิ่มฟีเจอร์ตามที่ตกลงกัน
3. รันคำสั่ง `npm run build` เพื่อตรวจสอบความถูกต้องของโค้ด
// turbo
4. รันคำสั่ง `npx cap sync` เพื่อซิงค์โค้ดไปยังโฟลเดอร์ Android/iOS
5. รายงานผลการติดตั้งและเสนอแนะขั้นตอนการทดสอบในลำดับถัดไป
