---
description: ขั้นตอนการเริ่มทำงานต่อหรือซิงค์ข้อมูลจาก GitHub
---

1. ตรวจสอบว่ามีการอัปเดตจาก GitHub หรือไม่โดยใช้คำสั่ง `git fetch`
2. แสดงรายการสิ่งที่เปลี่ยนแปลง (ถ้ามี) ให้ USER ทราบ
3. ดึงความเปลี่ยนแปลงล่าสุดลงมาด้วย `git pull origin main --rebase`
4. ตรวจสอบว่า Library ครบถ้วนหรือไม่ หากมีการอัปเดต package.json ให้รัน `npm install`
5. รันระบบทดสอบด้วย `npm run dev` เพื่อให้ USER พร้อมทำงานต่อ
